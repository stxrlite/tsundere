Templates

Templates are small starting points. They create the files needed to run, but they do not hide a full framework behind the project.

Create Syntax

tsundere create my-bot --template discord
tsundere create api --template rest
tsundere create socket --template websocket
tsundere create worker --template microservice
tsundere create tool --template cli
tsundere create scratch --template empty

Discord Bot

Creates a Discord client, ping command, command discovery config, `.env.example`, and the local `@tsundere/discord` runtime.

REST API

Creates a minimal Express health route. Use it when testing npm package compatibility or backend code.

WebSocket Server

Creates a small `ws` server starter. Useful for realtime experiments and bot dashboards.

Microservice

Creates a single async handler. Use it for jobs, queues, and small service modules.

CLI Tool

Creates a small command-line entry point. Good for local tools and scripts.

Empty Project

Creates the smallest possible `.yuri` project for testing compiler behavior.
