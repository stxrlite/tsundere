Getting Started

This page walks through installing Tsundere, creating a project, fixing existing projects, and understanding what files are generated.

Install From a Release Zip

Unzip the release folder and run the installer script. The installer checks Node.js, installs pnpm when missing, installs the Tsundere CLI globally, installs the VS Code or Cursor extension when `code` is available, verifies the CLI, and opens these docs.

.\install-tsundere.ps1

Create a Discord Project

tsundere create my-bot --template discord
cd my-bot
tsundere install
tsundere dev

The Discord template creates `src/main.yuri`, `src/commands/ping.yuri`, `.env.example`, `tsundere.config.json`, and a project-local `.tsundere/runtime/discord` package.

Add a Token

Copy `.env.example` to `.env`, then set your Discord token. The Tsundere runtime loads `.env` before starting your bot.

DISCORD_TOKEN=your-token-here

Fix an Existing Project

If Node says it cannot find `@tsundere/discord`, install the bundled runtime into that project.

tsundere runtime install
tsundere install

Your `package.json` should point to `file:.tsundere/runtime/discord`. After install, Node resolves `@tsundere/discord` from `node_modules` normally.

Starter Bot

import { Client, Intents, Slash, Embed } from "@tsundere/discord"

const client = new Client({
  token: env.DISCORD_TOKEN,
  intents: [Intents.Guilds, Intents.GuildMessages]
})

client.once("ready", () => {
  log(`Online as ${client.user.tag}`)
})

client.on("interactionCreate", async (interaction) => {
  if interaction.isCommand("ping") {
    await interaction.reply({
      embeds: [
        Embed.create()
          .title("Pong")
          .description(`Latency: ${client.ping}ms`)
          .color("#ff7ab6")
          .toJSON()
      ],
      ephemeral: true
    })
  }
})

Slash.command("ping")
  .description("Check bot latency")
  .register(client)

client.login()
