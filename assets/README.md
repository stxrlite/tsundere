# Tsundere Visual Assets

Place the official logo/profile image here:

```txt
assets/tsundere-logo.png
assets/tsundere-profile.png
```

Recommended source image:

- The chibi Tsundere mascot in the black hoodie with pink accents
- Full-body transparent or dark-background PNG
- Keep the original full-height file as `tsundere-profile.png`
- Export square/cropped versions as needed for package registries, GitHub, Discord, and VS Code

Recommended derivatives:

- `tsundere-logo-128.png`
- `tsundere-logo-256.png`
- `tsundere-logo-512.png`
- `tsundere-banner.png`

Do not redraw the mascot into syntax, APIs, or compiler messages. Use it only for branding surfaces.
