Discord Command Visualizer

Large bots get hard to navigate. The planned visualizer is a VS Code panel that shows commands, groups, subcommands, components, modals, events, and code relationships as a searchable graph.

Graph Shape

Admin
  Ban
  Kick
  Warn

  Timeout

  Service
  Database
  API

Click Navigation

Click `/admin ban`, a button handler, or a modal schema and jump directly to its source file.

Search

Search command names, custom IDs, modals, components, handlers, and related services.

Dead Command Detection

Warn when command code exists but never registers or never appears in command discovery metadata.

Duplicate Detection

Catch duplicate names, overlapping groups, and stale command state before deployment.
