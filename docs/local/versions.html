Versions and Updates

Tsundere can report the installed version, check GitHub releases, and automatically install the newest CLI release when you approve it.

Check Your Version

tsundere version

Check for Updates

tsundere updater
tsundere updater check
tsundere updater self --yes
tsundere updater self --dry-run

`tsundere updater self --yes` downloads the newest CLI tarball from the configured GitHub release and installs it globally with npm. `--dry-run` verifies the chosen release asset without changing the machine.

Configure the GitHub Repo

The package metadata points at `TsundereLang/tsundere`. During local testing you can override it with an environment variable.

$env:TSUNDERE_UPDATE_REPO = "owner/repo"
tsundere updater

Project Package Updates

If you pass package names, `tsundere update` behaves like a package update command instead of a CLI self-update check.

tsundere update discord.js
tsundere update mysql2
