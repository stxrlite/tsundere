Updates

This section explains what changed in the current local build and how to apply it to projects.

How to Update a Project

tsundere runtime install
tsundere install
tsundere build
tsundere start

How to Build a Release

npm run dist:release

Send the generated `release` folder to testers. They can run `installer.bat` or `install-tsundere.ps1`.

Planned Update Flow

Future versions should let `tsundere updater self` download the newest GitHub release, verify the package, install the CLI, refresh the VS Code extension, and reopen the local docs.

The current release checker is intentionally conservative. It tells you what version is available and where to download it without replacing files automatically.
