Compiler Plugin Marketplace

Tsundere should stay focused. Plugins live in a separate community repo so people can fork it, add examples, and open pull requests without bloating the core language package.

Plugin Repository

The official plugin registry repo is `TsundereLang/tsundere-plugins`.

git clone https://github.com/TsundereLang/tsundere-plugins
cd tsundere-plugins

To contribute, fork the repo, create a folder in `plugins/`, add a `plugin.json`, update `registry.json`, and open a pull request. Manual installs from GitHub links are supported through `tsundere plugin install`.

Plugin API Vision

Plugins should be able to hook parser output, AST transforms, type metadata, compiler emit, build pipeline steps, YuriLS behavior, CLI commands, diagnostics, and code generation.
